# yurifetch
Simple Gelbooru Image Fetcher written as C++ beginner practice

### Usage
```yurifetch "tag1+tag2+tags" <pid (optional)>```

### Installation
Write down your gelbooru API_KEY and USER_ID in api.h and run
``make && sudo make install`` to install it

### Dependencies
- libcurl
- pugixml
- viu (optional)
